# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it via:
- **GitHub:** Open a private security advisory at Security → Advisories → New

## Response Timeline

- Acknowledgement within **48 hours**
- Fix or mitigation within **7 days** for critical issues

## Scope

This is a static portfolio site. Relevant vulnerabilities include:
- XSS in blog rendering
- Exposed credentials or secrets
- Malicious dependencies
